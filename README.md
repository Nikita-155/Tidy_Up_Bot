# 🧹 TidyUp — Telegram-бот для автоматизации клининговой компании

TidyUp — это многофункциональный Telegram-бот, который объединяет клиентов, уборщиков и администрацию в единой автоматизированной платформе для быстрого и прозрачного управления заказами клининговых услуг.

---

## Возможности

### Для клиента
- Быстрое оформление заказа за 2 минуты
- Выбор нескольких типов уборки
- Выбор даты через интерактивный календарь
- Выбор временного слота
- Отправка номера телефона одной кнопкой
- Просмотр истории заказов
- Оценка работы уборщика после выполнения

### Для уборщика
- Авторизация по паролю
- Выход на смену
- Просмотр доступных заказов
- Взятие заказа с автоматической блокировкой для других
- Отметка о прибытии на место
- Отправка фотоотчета после уборки
- Просмотр активных заказов
- Статистика выполненных заказов и рейтинг

### Для администратора
- Авторизация по паролю
- Просмотр статистики (всего заказов, новых, завершенных)
- Просмотр всех заказов
- Просмотр списка уборщиков с их статусами и рейтингом
